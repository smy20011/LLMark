Installation
============

llmark requires Python ``>=3.8`` and can be installed from pypi via:

.. code-block:: bash

   python -m pip install llmark


To install directly from github, you can run:

.. code-block:: bash

   python -m pip install git+https://github.com/smy20011/LLMark.git

For development, its recommended to use Poetry:

.. code-block:: bash

   git clone https://github.com/smy20011/LLMark.git
   cd llmark
   poetry install
